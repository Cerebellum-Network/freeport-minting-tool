Preparation

Prepare folder with the following structure: 

|-- assets
    |-- asset1.jpg
    |-- asset2.mp4
    |-- ...
|-- previews
    |-- preview1.jpg
    |-- preview2.jpg
    |-- ...
|-- nfts.csv
|-- key.txt

Put assets to upload inside assets folder.

Put assets' previews inside previews folder.

Put your private key inside key.txt. How do I get it?

Set up NFTs in nfts.csv using the following template:

Name Description Supply Asset file name Preview file name Price (USD) Joint account Primary cut Secondary cut
Name 1 Desc 1 300 asset1.jpg preview1.jpg 40 0xbf420a26dfd9c5bbf8c04935bb9a7f12a5e3d84a 10000 100
Name 2 Desc 2 300 asset2.mp4 preview2.jpg 40 0xbf420a26dfd9c5bbf8c04935bb9a7f12a5e3d84a 10000 100
... ... ... ... ... ... ... ... ...

You can use Google Sheets to create a table and export it to CSV.

Important note: first row of the table considered as "title" row and ignored during the parsing.
